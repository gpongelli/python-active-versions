============
Installation
============


Stable release
--------------

To install python_active_versions, run this command in your terminal:

.. code-block:: console

    $ pip install python-active-versions

This is the preferred method to install python_active_versions, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for python_active_versions can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitlab.com/gpongelli/python-active-versions

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://gitlab.com/gpongelli/python-active-versions/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://gitlab.com/gpongelli/python-active-versions
.. _tarball: https://gitlab.com/gpongelli/python-active-versions/tarball/master
